# ERP沙盘产能&原料预测工具 - 技术规格说明书

## 一、项目概述

**项目名称**: ERP沙盘产能&原料预测工具
**项目类型**: 纯前端单页面应用
**核心功能**: 生产线产能递推、原材料需求测算、可视化数据展示、AI智能分析
**技术栈**: HTML5 + CSS3 + JavaScript (原生ES6+) + Chart.js + 无后端依赖

| 生产线类型 | 生产周期(季度) |
|-----------|---------------|

|------|---------|

| P2   | 1R1 + 1R2 |

### 2.3 原材料采购提前期
| 原材料 | 采购提前期 |
|--------|-----------|
| R1     | 1季度     |
| R2     | 1季度     |
| R3     | 2季度     |
| R4     | 2季度     |

## 三、核心计算逻辑

### 3.1 季度产能递推规则 (Q1-Q4循环递推)

**规则1**: 生产线剩余周期=1 → 本季度产出1件产品，完工后立即重启生产，重置为该生产线标准生产周期

**规则2**: 生产线剩余周期>1 → 本季度无产出，剩余周期自动-1，继续生产

**规则3**: 生产线剩余周期=0 → 产线空闲，本季度无产出，立即开工，重置为标准生产周期

### 3.2 原材料需求计算

基于「季度产品产量 × 对应产品BOM用量」矩阵求和，计算R1/R2/R3/R4分季度需求量，结合提前期标注采购风险。

## 四、页面模块结构

### 模块1: 参数配置模块
- 展示沙盘默认参数(生产线周期、BOM结构、采购提前期)
- 支持用户手动修改参数
- 提供【一键恢复默认参数】按钮

### 模块2: 生产线数据录入模块
- 新增/删除多条生产线
- 录入项: 线号、生产线类型(下拉)、当前生产产品(下拉)、剩余生产周期(数字输入)
- 系统自动匹配生产周期

### 模块3: 自动测算核心模块
- 季度产能递推计算
- 单品产量汇总
- 原材料需求拆解
- 全年数据统计

### 模块4: 可视化数据看板模块
- 数据表格: 生产线季度明细、产品季度产能表、原材料季度需求表
- 数据图表: 产品全年产能柱状对比图、原材料需求占比饼图
- 核心指标卡片: 全年总产能、最高产能产品、原料总需求、紧缺原料

### 模块5: AI智能分析模块
- 用户配置API信息(地址、密钥)
- 一键触发AI分析
- 自动打包全部产线状态、产能数据、物料需求数据
- 返回标准化沙盘分析内容

## 六、技术实现要点

### 6.1 数据存储
- 所有数据存储在JavaScript内存对象中
- 无需localStorage或任何持久化存储
- 重置功能清空所有输入数据

### 6.2 图表库
- 使用Chart.js 4.x CDN
- 柱状图: 产品产能对比
- 饼图: 原材料需求占比

### 6.3 AI API调用
- 前端直接POST到用户配置的API地址
- 请求体JSON格式包含完整业务数据
- 支持流式响应解析(如SSE)

### 6.4 计算引擎
- 纯JavaScript实现
- 季度递推循环4次
- 每条生产线独立计算
